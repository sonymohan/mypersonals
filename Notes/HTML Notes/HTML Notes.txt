The DOCTYPE indicates the standards compliance or version of the document

sample:
	<!DOCTYPE HTML>

Strict HTML 4.01 doctype: 
	<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
	"http://www.w3.org/TR/html4/strict.dtd">
	
The HTML element acts as the root container of the document
<html>
	----Page content-----
</html>

Inside the HTML element we find 2 other core elements: HEAD and BODY

The HEAD element contains document metadata
The BODY element contains the content to be displayed

In the <head> there are 6 key other elements listed:
1) <title>
2) <meta>
3) <script>
4) <link>
5) <style>
6) <base>

In the <body> the below elements are used as core elements while creating Web pages:
1) Text
2) Lists
3) Links
4) Tables
5) Images
